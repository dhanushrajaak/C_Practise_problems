# C Programming Practice Repository

This repository contains C programming exercises, algorithm implementations, coding interview questions, and problem-solving programs completed as part of continuous learning and practice.

The purpose of this repository is to strengthen programming fundamentals, improve logical thinking, and develop efficient solutions to commonly asked coding problems and technical interview questions.

## Objectives

* Strengthen C programming fundamentals.
* Improve problem-solving and analytical skills.
* Practice coding interview and LeetCode-style questions.
* Gain a deeper understanding of algorithms and programming concepts.
* Build a well-organized collection of reference implementations.

## Environment

* Language: C
* Compiler: GCC
* Platform: Linux 

## Compilation

```bash
gcc program_name.c -o output
```

## Execution

```bash
./output
```

## Notes

* All programs are written for learning, practice, and reference purposes.
* Solutions focus on correctness, readability, and understanding of core concepts.
* The repository is regularly updated with new programs and problem-solving approaches.
